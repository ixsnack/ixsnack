# module import

module 의 import 하기

&nbsp;
&nbsp;

## import
```js
import $ from 'jquery'
import 'ixsnack'

$(target).ixSlideMax()
```

&nbsp;
&nbsp;

## require
```js
const $ = require('jquery')
const ixsnack = require('ixsnack')

$(target).ixSlideMax()
```
